# radiotalk

## Datasets

- [twangodev/radiotalk-voices-2k](https://huggingface.co/datasets/twangodev/radiotalk-voices-2k) — 2,000 English reference voices for TTS voice cloning.
- [twangodev/radiotalk-us-transcripts-100k](https://huggingface.co/datasets/twangodev/radiotalk-us-transcripts-100k) — 100k synthetic US ATC transcripts.
